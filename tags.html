Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings Defines HTML headings
p paragraph Defines a paragraph
table table Defines a table
tr table row Defines a row in a table
th table header Defines a header cell in a table
td table data Defines a cell in a table
div content division Defines a section in a document
header header Defines a header for a document or section
ol ordered list Defines an ordered list
ul unordered list Defines an unordered list
li list item Defines a list item
blockquote blockquote Defines a section that is quoted from another source
br line break Defines a single line break

Inline Elements

Tag Name Purpose Example Result
em emphasized Defines emphasized text <em>technical sophistication</em> technical sophistication
strong strong Defines important text <strong>at least a billion people</strong> at least a billion people
a anchor Defines a hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image Defines an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Defines a piece of computer code <code>$ ls -hartl</code> $ ls -hartl
span span Defines a section in a document There were once <span>Three Musketeers<span> There were once Three Musketeers

Document Tags

Tag Purpose
html Defines the root of an HTML document
head Defines information about the document
body Defines the document's body
title Defines a title for the document
meta Defines metadata about an HTML document
